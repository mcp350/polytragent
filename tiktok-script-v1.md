## Psychology Frameworks Used

1. **Loss Aversion**: "If you're trading without this, you're flying blind" — frames NOT using it as losing
2. **Purple Cow Hook**: "Printing money and nobody's talking about it" — remarkable claim that demands attention
3. **LF8 #6 Superiority**: Positions user as smarter than other traders by using AI research
4. **LF8 #3 Freedom from fear**: "Stop guessing" removes the anxiety of uncertainty
5. **Scarcity/FOMO**: "Go try it before this blows up" — implies limited window
6. **Authority**: Showing actual bot output on screen = credible proof
7. **Social Proof**: "I've been using it for two weeks" = personal testimonial
8. **Processing Fluency**: Short sentences, simple language, visual proof on screen
